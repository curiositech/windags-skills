# Orchestrator (DEPRECATED)

**This skill has been merged into `dag-orchestrator`.**

The orchestrator's coordination capabilities have been combined with dag-executor's execution capabilities to create a unified, more powerful skill.

## Migration

Use `dag-orchestrator` instead, which provides:

- All orchestration capabilities from this skill
- Wave-based parallel execution from dag-executor
- HTDAG (Hierarchical Task DAG) architecture
- As-needed decomposition (ADaPT algorithm)
- Dynamic node expansion
- Phase orchestration for complex multi-stage tasks

## Activation

If you meant to invoke orchestration capabilities, use:

```
"orchestrate this task using dag-orchestrator"
"execute DAG for: <task>"
"decompose and coordinate: <complex task>"
```
