# Evaluation Report Template

Use this template for experiment preregistration, final reporting, or internal evaluation memos.

## Research Question

State one falsifiable sentence.

## Method

- Systems under test and versions
- Scenarios and sampling strategy
- Metrics with exact definitions
- Sample-size justification
- Procedure, randomization, and stopping rule

## Results

- Descriptive stats with spread and confidence intervals
- Test statistics and p-values
- Effect sizes with confidence intervals
- Inter-rater reliability, if human evaluation is involved

## Discussion

- Practical significance, not just statistical significance
- Null results reported honestly
- What changed your interpretation during analysis

## Threats to Validity

- Internal
- External
- Construct
- Statistical conclusion

## Reproduction

- Code and data location
- Exact software versions
- Hardware and network context
